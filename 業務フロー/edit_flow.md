```
@startuml

opt マイページ

ユーザー -> Webサーバー : 投稿記事選択
Webサーバー-> DBサーバー : 選択された投稿記事を送信
DBサーバー -> Webサーバー : 選択された投稿記事を送信
Webサーバー -> ユーザー : 投稿記事の表示

opt アカウントの情報を変更するとき

ユーザー -> Webサーバー : 編集したアカウント情報送信
Webサーバー-> DBサーバー : アカウント情報を送信
DBサーバー --> DBサーバー : アカウント情報を保存
DBサーバー --> Webサーバー : 保存したアカウント情報送信
Webサーバー -> ユーザー : アカウント情報表示

end

opt 記事(コメント)変更するとき

ユーザー -> Webサーバー : 編集した投稿記事送信
Webサーバー-> DBサーバー : 変更記事を送信
DBサーバー --> DBサーバー : 投稿記事を保存
DBサーバー --> Webサーバー : 保存した記事送信
Webサーバー -> ユーザー : 変更した記事表示

end

opt 記事(コメント)を削除するとき

ユーザー -> Webサーバー : 選択した記事削除
Webサーバー-> DBサーバー : 選択した記事を送信
DBサーバー --> DBサーバー : 選択された記事を削除
DBサーバー --> Webサーバー : 投稿記事を送信
Webサーバー -> ユーザー : 投稿記事表示


end
end

@enduml
```
