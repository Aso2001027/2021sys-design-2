```
@startuml

opt 記事削除

管理者 -> Webサーバー : 削除する記事を選択
Webサーバー -> DBサーバー : 削除する記事の情報を送信
DBサーバー --> DBサーバー : 削除した記事の情報を保存
DBサーバー --> Webサーバー : 記事削除後の情報を送信
Webサーバー -> 管理者 : 記事削除後の全ての記事を一覧表示

end
opt アカウント削除

管理者 -> Webサーバー : 削除するアカウントを選択
Webサーバー -> DBサーバー : 削除するアカウントの情報を送信
DBサーバー --> DBサーバー : 削除したアカウントの情報を保存
DBサーバー --> Webサーバー : アカウント削除後の情報を送信
Webサーバー -> 管理者 : アカウント削除後の全てのアカウント一覧を表示

end

@enduml
```
