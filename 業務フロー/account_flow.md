```
@startuml

opt 共有する情報

ユーザー -> Webサーバー : 共有する記事を選択
Webサーバー -> DBサーバー : 共有する記事を送信
DBサーバー --> DBサーバー : 選択した情報の保存
DBサーバー --> Webサーバー : 保存した情報を送信
Webサーバー -> ユーザー : 保存した情報一覧の表示

end
opt 報告する処理

ユーザー -> Webサーバー : 報告する記事選択
Webサーバー -> DBサーバー : 選択した情報を送信
DBサーバー --> DBサーバー : 選択した情報の保存
DBサーバー --> Webサーバー : 保存した情報を送信
Webサーバー -> ユーザー : 報告数を表示

end

@enduml
```
