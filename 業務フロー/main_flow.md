```
@startuml

ユーザー -> Webサーバー : 検索
Webサーバー -> DBサーバー : 検索結果の要求
DBサーバー --> Webサーバー : 検索結果を送信
Webサーバー -> ユーザー : 検索結果記事一覧の表示

opt 投稿表示

ユーザー -> Webサーバー : 投稿選択
Webサーバー -> DBサーバー : 投稿情報の要求
DBサーバー --> DBサーバー : 選択した情報の保存
DBサーバー --> Webサーバー : 投稿情報の送信
Webサーバー -> ユーザー : 投稿情報の表示

opt お気に入り追加するとき

ユーザー -> Webサーバー : 選択(お気に入りにする投稿)
Webサーバー-> DBサーバー : 投稿情報を送信
DBサーバー --> DBサーバー : 選択した情報の保存
DBサーバー --> Webサーバー : 保存したい情報を送信(お気に入り)
Webサーバー -> ユーザー : お気に入り情報の表示

end

opt コメントするとき

ユーザー -> Webサーバー : コメント送信
Webサーバー-> DBサーバー : コメント送信
DBサーバー --> DBサーバー : 投稿情報に対するコメント保存
DBサーバー --> Webサーバー : 情報に対するコメント送信
Webサーバー -> ユーザー : 投稿に対するコメント表示

end

end

opt 投稿するとき
ユーザー -> Webサーバー : 投稿する情報の選択
Webサーバー-> DBサーバー : 投稿する情報の送信
DBサーバー --> DBサーバー : 投稿する情報を保存
DBサーバー --> Webサーバー : 保存した情報を送信
Webサーバー -> ユーザー : 保存した情報の表示

end

ユーザー -> Webサーバー : 相手情報の選択
Webサーバー-> DBサーバー : 相手情報の送信
DBサーバー --> DBサーバー : 相手情報を保存
DBサーバー --> Webサーバー : 保存した情報を送信
Webサーバー -> ユーザー : 保存した情報の表示


@enduml
```
