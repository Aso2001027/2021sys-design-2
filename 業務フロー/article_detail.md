```
@startuml
opt お気に入り追加するとき

ユーザー -> Webサーバー : 選択(お気に入りにする投稿)
Webサーバー-> DBサーバー : 投稿情報を送信
DBサーバー --> DBサーバー : 選択した情報の保存
DBサーバー --> Webサーバー : 保存したい情報を送信(お気に入り)
Webサーバー -> ユーザー : お気に入り情報の表示

end

opt コメントするとき

ユーザー -> Webサーバー : コメント送信
Webサーバー-> DBサーバー : コメント送信
DBサーバー --> DBサーバー : 投稿情報に対するコメント保存
DBサーバー --> Webサーバー : 情報に対するコメント送信
Webサーバー -> ユーザー : 投稿に対するコメント表示

end
 
opt 共有する処理

ユーザー -> Webサーバー : 共有ボタンを選択
Webサーバー -> ユーザー : 共有先にURLを送り、共有先へ遷移

end
opt 報告する処理

ユーザー -> Webサーバー : 報告する記事選択
Webサーバー -> DBサーバー : 選択した情報を送信
DBサーバー --> DBサーバー : 選択した情報の保存
DBサーバー --> Webサーバー : 保存した情報を送信
Webサーバー -> ユーザー : 報告数を表示

end

opt 関連記事表示

ユーザー -> Webサーバー : カテゴリ、タグの情報を送信
Webサーバー -> DBサーバー : カテゴリ、タグの情報を送信
DBサーバー --> Webサーバー : カテゴリ、タグが一致した記事の情報を送信
Webサーバー -> ユーザー : カテゴリ、タグが一致した記事を表示

end
@enduml
```
